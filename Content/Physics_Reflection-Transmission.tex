reflection  amplitude coefficients:

$
a_{r\parallel} = 
\frac{ \mathscr{E}_{0,r\parallel} }{ \mathscr{E}_{0,\alpha\parallel} } = 
- \frac{sin(\alpha - \beta)}{sin(\alpha + \beta)}
$

$
a_{r\perp} = 
\frac{ \mathscr{E}_{0,r\perp} }{ \mathscr{E}_{0,\alpha\perp} } = 
\frac{tan(\alpha - \beta)}{tan(\alpha + \beta)}
$

\

Reflection:

$
A_r = 
\frac{P_r}{P_\alpha} = 
\frac{a_{r\perp}^2 \mathscr{E}^2_{0,\alpha\perp} + a_{r\parallel}^2 \mathscr{E}^2_{0,\alpha\parallel} }{\mathscr{E}^2_{0,\alpha\perp} + \mathscr{E}^2_{0,\alpha\parallel}}
$

\

transmission amplitude coefficients:

$
a_{t\parallel} = 
\frac{ \mathscr{E}_{0,t\parallel} }{ \mathscr{E}_{0,\alpha\parallel} } = 
2 \frac{sin(\alpha)sin(\beta)}{sin(\alpha + \beta)}
$
 
$
a_{t\perp} = 
\frac{ \mathscr{E}_{0,t\perp} }{ \mathscr{E}_{0,\alpha\perp} } = 
2 \frac{sin(\alpha)sin(\beta)}{sin(\alpha + \beta)cos(\alpha - \beta)}
$

\

Transmission:

$
A_t =  
\frac{P_t}{P_\alpha} = 
\frac{a_{t\perp}^2 \mathscr{E}^2_{0,\alpha\perp} + a_{t\parallel}^2 \mathscr{E}^2_{0,\alpha\parallel} }{\mathscr{E}^2_{0,\alpha\perp} + \mathscr{E}^2_{0,\alpha\parallel}}
$

\

$ A_r + A_t = 1 $
